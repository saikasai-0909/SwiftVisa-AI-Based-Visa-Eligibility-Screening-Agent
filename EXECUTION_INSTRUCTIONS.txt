================================================================================
                    üîß TROUBLESHOOTING
================================================================================

PROBLEM 1: "streamlit: command not found"
------------------------------------------
SOLUTION:
pip install streamlit
Or use full path:
python -m streamlit run visa_form_app.py

PROBLEM 2: "Module 'sentence_transformers' not found"
------------------------------------------------------
SOLUTION:
pip install sentence-transformers

PROBLEM 3: "Cannot connect to LM Studio"
-----------------------------------------
ERROR MESSAGE:
‚ö†Ô∏è Error: Connection refused
‚ö†Ô∏è Please ensure LM Studio is running on http://localhost:1234

SOLUTION:
1. Open LM Studio
2. Load model: meta-llama-3.2-3b-instruct
3. Click "Start Server"
4. Verify it says "Server running on port 1234"
5. Restart application

PROBLEM 4: "File does not exist: visa_faiss.index"
---------------------------------------------------
SOLUTION:
Check file location:
dir visa_faiss.index

If missing, you need to rebuild:
python create_embeddings.py
python store_in_vector_db.py

PROBLEM 5: "Port already in use"
---------------------------------
ERROR MESSAGE:
Address already in use

SOLUTION:
Use different port:
python -m streamlit run visa_form_app.py --server.port 8502

Or kill existing process:
netstat -ano | findstr :8501
taskkill /PID [PID_NUMBER] /F

PROBLEM 6: Application loads but shows errors
----------------------------------------------
SOLUTION:
1. Check LM Studio is running
2. Verify model is loaded
3. Check console for error messages
4. Ensure all files exist (4 critical files)
5. Restart application

PROBLEM 7: Slow response or timeout
------------------------------------
SOLUTION:
1. LLM might be loading (wait 30 seconds)
2. First query is always slower
3. Check LM Studio hasn't crashed
4. Reduce query complexity
5. Restart LM Studio
