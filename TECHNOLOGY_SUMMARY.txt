PROJECT NAME: AI SwiftVisa - UK Visa Eligibility Checker
--------------------------------------------------------------------------------

TECHNOLOGIES USED:
================================================================================

1. PROGRAMMING LANGUAGE
   ---------------------
   - Python 3.12.0

2. WEB FRAMEWORK (UI)
   -------------------
   - Streamlit - For creating interactive web forms and user interface

3. MACHINE LEARNING / AI
   ----------------------
   - Sentence Transformers (all-MiniLM-L6-v2) - For text embeddings
   - LLaMA 3.2 3B Instruct - Large Language Model for response generation

4. VECTOR DATABASE
   ----------------
   - FAISS (Facebook AI Similarity Search) - For storing and searching embeddings

5. API FRAMEWORK
   --------------
   - OpenAI API (Compatible) - Communication with LM Studio

6. LLM HOSTING
   ------------
   - LM Studio - Local LLM inference server (http://localhost:1234)

7. DATA FORMAT
   ------------
   - NumPy (.npy files) - For embedding storage
   - Pickle (.pkl files) - For metadata storage
   - JSONL files - For document metadata
